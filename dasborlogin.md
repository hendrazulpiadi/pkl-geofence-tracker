# LOGIN PAGE DESIGN SPECIFICATION

## Sistem Informasi Absensi PKL (SIA PKL)

### ROLE

You are a Senior Product Designer, SaaS UX Architect, and Frontend UI Expert.

Design a modern, professional, and highly usable Login Page for a Web-Based Internship Attendance System (SIA PKL).

The target users are:

* SMK Students
* School Mentors
* Company Mentors
* System Administrators

The interface must feel modern, trustworthy, easy to use, and optimized for mobile devices.

---

# DESIGN GOALS

The login page should communicate:

* Professionalism
* Trust
* Simplicity
* Speed
* Real-Time Attendance Verification

Users should immediately understand that this system uses:

* GPS Verification
* Selfie Verification
* Real-Time Attendance Monitoring

* Modern SaaS Dashboard Style
* Clean UI
